# Installation
{: .no_toc }

This section describes the steps required to include the Autosuggest JS SDK and its default styles in your application.

## Prerequisite

Please complete the Self Serve FTU (First-Time User) flow for sign up, site creation, feed upload, and relevancy configuration.

[Self Serve Dashboard Documentation](https://unbxd.com/docs/site-search/integration-documentation/onboarding-flow/)

---

## Include the SDK

Include the Autosuggest JS SDK in your HTML file using the following script tag:

    <script src="https://libraries.unbxdapi.com/search-sdk/v1/autosuggest.min.js"></script>

---

## Include Default Styles

To apply the default Autosuggest UI styles, include the SDK stylesheet:

    <link rel="stylesheet" href="https://libraries.unbxdapi.com/search-sdk/v1/autosuggest.min.css" />
